# happ-haskell

## Prerequisites

- ghc
- stack

## Getting started

```bash
# Clone sources
$ http://github.com/korczis/happ.git

# Go to folder with happ-haskell sources
$ cd happ/happ-haskell

# Build project using stack
$ stack build
```

## Running

***Run "hello world" program***

```bash
# Run program using stack
$ stack run hello-world
```
