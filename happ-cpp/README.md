# happ-cpp

## Prerequisites

- C/C++ compiler
- CMake

## Getting started

```bash
# Clone sources
$ http://github.com/korczis/happ.git

# Go to folder with happ-cpp sources
$ cd happ/happ-cpp

# Create build folder
$ mkdir build

# Enter build folder
$ cd build

# Run CMake
$ cmake ..
```
